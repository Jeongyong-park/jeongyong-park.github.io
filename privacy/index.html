개인정보처리방침

최종 업데이트: 2025년 6월 4일

1. 개요​

쌍팔년생 개발자 블로그(이하 "본 사이트")는 방문자의 개인정보 보호를 중요하게 생각합니다. 본 개인정보처리방침은 본 사이트가 수집하는 정보, 사용 방법, 보호 조치에 대해 설명합니다.

2. 수집하는 정보​

2.1 자동으로 수집되는 정보​

  • 방문 기록: IP 주소, 브라우저 정보, 운영체제, 접속 시간
  • 사용 패턴: 방문한 페이지, 체류 시간, 클릭 경로
  • 기기 정보: 화면 해상도, 언어 설정

2.2 쿠키 및 유사 기술​

본 사이트는 다음 목적으로 쿠키를 사용합니다:

  • 사이트 기능 개선 및 사용자 경험 향상
  • 웹사이트 트래픽 분석
  • 광고 개인화 (Google AdSense)

2.3 댓글 및 연락처​

  • 댓글 작성 시: 이름, 이메일 주소 (선택적)
  • 연락 시: 제공하신 연락처 정보

3. 정보 사용 목적​

수집된 정보는 다음 목적으로 사용됩니다:

  • 웹사이트 서비스 제공 및 개선
  • 사용자 경험 최적화
  • 사이트 트래픽 분석
  • 법적 요구사항 준수
  • 보안 및 사기 방지

4. 제3자 서비스​

4.1 Google Services​

  • Google Analytics: 웹사이트 트래픽 분석
  • Google AdSense: 광고 표시
  • Google Fonts: 웹 폰트 제공

4.2 호스팅 및 CDN​

  • GitHub Pages: 웹사이트 호스팅
  • CDN 서비스: 콘텐츠 전송 최적화

4.3 댓글 시스템​

  • Utterances/Giscus: GitHub 기반 댓글 시스템

5. Google AdSense 및 광고​

본 사이트는 Google AdSense를 통해 광고를 표시합니다:

  • Google은 쿠키를 사용하여 관련성 높은 광고를 표시합니다
  • 방문자는 Google 광고 설정에서 개인화 광고를 거부할 수 있습니다
  • DoubleClick DART 쿠키에 대한 자세한 내용은 Google 개인정보처리방침을 참조하세요

6. 쿠키 관리​

6.1 쿠키 유형​

  • 필수 쿠키: 사이트 기본 기능 제공
  • 분석 쿠키: 사이트 사용 패턴 분석
  • 광고 쿠키: 개인화된 광고 표시

6.2 쿠키 거부​

브라우저 설정을 통해 쿠키를 거부할 수 있으나, 일부 사이트 기능이 제한될 수 있습니다.

7. 데이터 보안​

개인정보 보호를 위해 다음 조치를 취합니다:

  • HTTPS 암호화 통신
  • 정기적인 보안 업데이트
  • 접근 권한 제한
  • 데이터 최소 수집 원칙

8. 데이터 보관 및 삭제​

  • 방문 기록: 최대 26개월 보관 (Google Analytics 기준)
  • 댓글 데이터: GitHub 저장소에 영구 보관
  • 요청 시 삭제: 연락을 통해 개인정보 삭제 요청 가능

9. 아동의 개인정보​

13세 미만 아동의 개인정보는 의도적으로 수집하지 않습니다. 아동의 정보가 수집된 것을 발견하면 즉시 삭제합니다.

10. 국제 데이터 전송​

본 사이트는 전 세계 사용자에게 서비스를 제공하며, 데이터는 다양한 국가의 서버에 저장될 수 있습니다.

11. 개인정보처리방침 변경​

본 방침의 중요한 변경사항은 사이트에 공지하며, 지속적인 사이트 이용은 변경된 방침에 대한 동의로 간주됩니다.

12. 연락처​

개인정보처리방침에 대한 문의사항이 있으시면 다음으로 연락해 주세요:

  • 이메일: kladess@gmail.com
  • GitHub: @jeongyong-park

13. 권리 행사​

다음 권리를 행사할 수 있습니다:

  • 개인정보 처리 현황 확인
  • 개인정보 수정 및 삭제
  • 개인정보 처리 정지 요구
